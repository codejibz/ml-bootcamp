# ML Bootcamp

A 60-day intensive project-based machine learning engineering bootcamp.

## Progress
- Start date: May 2026
- Current week: Week 0 — Environment Setup

## Stack
- Python 3.12.7
- NumPy, Pandas, Matplotlib, Seaborn, scikit-learn, Jupyter
- PyTorch (Week 5+)

## Structure
- `notebooks/` — exploration notebooks (numbered)
- `src/` — reusable Python modules
- `data/` — raw and processed datasets
- `models/` — serialized model artifacts
- `reports/` — figures and findings
- `tests/` — unit tests (Week 6+)
